# Wavelength — the station on your wavelength

Wavelength turns your interests, today's news, and your music taste into a live personal radio show. Gemini (with Google Search grounding) writes a show around what you care about *right now* — including major events happening today — and Gemini TTS voices your pick of four original hosts. **Lyria, DeepMind's music model, composes each listener a personal station ident on the spot.** Music comes from your own YouTube likes (played through a visible, ToS-compliant embedded player) or bundled royalty-free beds, with radio-style ducking and crossfades. Skip a track and the host reacts on air, instantly. Advertisers get a self-serve booth where Gemini rewrites their approved facts into a personal 8-second sponsor read for every single listener. One listener, one station: **"\<YourName\> FM"**.

## What makes it personal

- **Station fingerprint** — Google sign-in + optional YouTube consent; Gemini distills your liked videos & subscriptions into 5 editable interest chips. No watch history, no Gmail, no mining.
- **Today, not someday** — Google Search grounding pulls real stories from the last days, and finals/launches happening *today* lead the show.
- **Your weather, your hour** — with location consent, the host opens with the actual sky outside and matches the daypart's energy (Open-Meteo, keyless).
- **Link deep-dives** — paste any article URL; the host reads it (Gemini URL context) and discusses its actual content on air, with prompt-injection defenses.
- **Four hosts, one dial** — Nova (late-night warmth), Riff (6AM broadcast hurricane), RJ Meethi (Mumbai ki pyaari awaaz, Hinglish), Velvet ('70s soul swagger). All original personas — never imitations of real people — each with its own Gemini TTS voice and delivery.
- **Live reactions** — skip ("not tonight") or love ("more like this") any track; pre-generated in-persona reactions play with zero lull.
- **Your Signature Theme** — Lyria composes a ~30s instrumental ident from your fingerprint, cached per listener so it's yours.

Built for the Stanford x DeepMind hackathon (Replit track). Next.js 16 App Router · TypeScript · Tailwind v4 · `@google/genai`.

## Architecture

```
┌────────────────────────────── Browser ──────────────────────────────┐
│  Landing → Tune (edit interest chips) → Live (Web Audio player)     │
│  Optional: Google Sign-In + YouTube likes/subs (client-side, OAuth) │
└──────┬──────────────────────────────────────────────────────────────┘
       │ POST /api/profile   liked/sub titles → 5 interest chips
       │ POST /api/show      interests → Show JSON (talk/song segments)
       │ POST /api/tts       talk text → Nova's voice (WAV)
       │ GET/POST /api/ads   self-serve ad campaigns
       │ POST /api/track · GET /api/stats   metrics for /admin
┌──────▼──────────────── Next.js 16 (Node runtime) ───────────────────┐
│  lib/gemini.ts   text model + Google Search grounding + TTS,        │
│                  hardcoded fallbacks — the demo never 500s          │
│  lib/store.ts    JSON store at data/store.json (stats, campaigns)   │
│  lib/wav.ts      PCM → RIFF/WAV for TTS audio                       │
│  public/music/   synth beds generated by scripts/make-music.mjs     │
└─────────────────────────────────────────────────────────────────────┘
```

## Run it on Replit in 5 minutes

1. **Import** this repo into Replit (Create Repl → Import from GitHub). The `.replit` file already configures Node 22, dev run, and deployment.
2. **Secrets**: add `GEMINI_API_KEY` (from [Google AI Studio](https://aistudio.google.com/apikey)). That's the only required secret — everything else has defaults (see `.env.example`).
3. **Optional — Google sign-in + YouTube taste import**: create an OAuth Web client in Google Cloud Console, add your Repl URL to *Authorized JavaScript origins*, enable the YouTube Data API v3, and set `NEXT_PUBLIC_GOOGLE_CLIENT_ID`. Without it the app runs in Demo Mode.
4. **Run**. Open the webview → *Try demo* → tweak your chips → **Go Live**.
5. Visit `/advertise` to book an ad and `/admin` to watch the numbers move.

Local dev is the same: copy `.env.example` to `.env.local`, fill in the key, `npm run dev`. Regenerate music beds anytime with `node scripts/make-music.mjs` (needs `ffmpeg` on PATH), or drop your own mp3s into `public/music/`.

## Safety notes

- **No YouTube downloads.** No audio downloading or extraction, no watch-history access — liked videos and subscriptions are read client-side with the user's own OAuth token. Playback of the user's music happens exclusively through a **visible embedded YouTube player** (IFrame API, on-screen, volume-faded via the official API); non-embeddable videos are skipped, never worked around.
- **Original hosts.** All four personas are invented characters and never imitate real presenters; voices are stock Gemini TTS voices. Fetched web/search content is treated as material to discuss — embedded instructions in pages are explicitly disregarded.
- **Ad transparency.** Every ad read starts with an explicit sponsor disclosure ("Today's show is brought to you by …"), copy is generated strictly from advertiser-approved `facts`, and medical/financial/gambling/political campaigns are rejected.
- **Never fail the demo.** Every Gemini call has a canned fallback; missing keys degrade gracefully instead of erroring.
